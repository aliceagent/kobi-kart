# Kobi Kart

A bright, couch-multiplayer kart racer for the browser. Two players share one
keyboard and race a 4-track Grand Prix against AI opponents on procedurally
generated tracks, with power-ups, guard rails, themed worlds, and a festive
award ceremony.

Built with [Phaser 3](https://phaser.io/) + [Vite](https://vitejs.dev/).

## Play

- **Player 1:** `A`/`D` steer · `S` brake/drift · `W` boost · `Shift` use item
- **Player 2:** `←`/`→` steer · `↓` brake/drift · `↑` boost · `Enter` use item
- **`M`** mutes audio

## Develop

```bash
npm install
npm run dev      # local dev server
npm run build    # production build into dist/
npm run preview  # preview the production build
```

## Worlds

Grassy · Beach · Ice · Candy — each with its own palette, props, obstacles,
and themed music (chiptune arrangements of public-domain classics).
